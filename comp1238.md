## Week 1 - History
- Computers were developed over many years
- Early computers were very large and slow

## Week 2 - Bits and Bytes
- RAM is short-term memory while storage is persistent
- Data in computers is represented as sequences of bytes
- ASCII table represents characters in memory

## Week 3 - Text Files and Markup
- Structure of a text file includes punctuation and line breaks
- Markdown and HTML are markup languages
- JSON and XML are structured data formats

## Week 4 - Typing and Searching
- Keyboard shortcuts and touch typing improve speed and productivity
- Text editors include Notepad, vi, and VSCode
- Wildcards and regular expressions help searching in text.

## Week 5 - Command Line and SSH
- Command Line Interface (CLI) basics and its importance
- Difference between CLI, Terminal, and Shell
- Using SSH for remote access to servers
- Transferring files using sftp and scp

## Week 6 - Git and Versioning
- Using diff tools to compare file changes
- Basic Git workflow (add, commit, push)
- Working with GitHub repositories
